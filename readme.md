El objetivo de este proyecto es aprender a usar los bloques de construcción más importantes de JavaScript paso a paso.


el proyecto en tu navegador, ocurren las siguientes cosas en orden:
1 *te da la bienvenida:*
2 *te pide datos:* Te solicita que escribas dos números.
3 *revisa que todo esté bien:* Si por error escribes letras en lugar de números, la aplicación se da cuenta, te avisa del error y cambia los valores a 0 para que el programa no se rompa.
4 *hace los cálculos:** Suma, resta, multiplica y divide esos números automáticamente.
5 *analiza los resultados:* Te dice si los resultados son grandes, pequeños o si dan un número exacto.
6 *guarda un historial:** Organiza todas las operaciones en listas y te muestra un resumen detallado en la consola.


El código está organizado en 5 lecciones, tal como se aprende en el curso:

Lección #1: El primer contacto
Aquí el programa saluda. usamos `alert()` para mostrar la ventana emergente en la pantalla y `console.log()` para escribir textos "ocultos" que solo se ven en la herramienta de desarrollador del navegador.

Lección #2: guardar cajas (Variables) y tomar decisiones
*Guardar datos:* Usamos `let` para crear "cajas virtuales" donde guardamos los dos números que escribiste.
*Tomar decisiones (Condicionales):* usamos herramientas llamadas `if` (si pasa esto...) y `switch` (en el caso de que el número sea...) para que el programa responda de forma diferente según el resultado de los cálculos.

Lección #3: Listas (Arreglos) y Repeticiones (Ciclos)
*Ciclos (`for` y `while`):* Son motores que repiten una acción varias veces de forma automática (en el código los usamos para contar del 1 al 5).
*Arreglos:** Son como una caja con compartimentos para guardar muchos datos juntos. Creamos una lista llamada `listaResultados` para guardar los 4 resultados matemáticos.

Lección #4: Máquinas de cálculo (Funciones)
Para no repetir código, creamos funciones (`sumar`, `restar`, etc.). Cada una recibe dos números, hace su trabajo y te devuelve el resultado. 
*Protección:* La máquina de dividir tiene una regla especial: si intentas dividir por 0 (lo cual es imposible en matemáticas), te pone un cartel de advertencia y te devuelve 0.

Lección #5: Organizar los datos como en la vida real (Objetos)
Un objeto en programación es una estructura que guarda características de algo en un solo lugar. Creamos un objeto llamado `operacionMatematica` que guarda los números que pusiste, sus resultados y una función interna (un método) que imprime un resumen ordenado de todo lo que pasó.
